Retail Sales Dashboard
Overview

This project is an interactive Power BI dashboard built using the Superstore dataset. It provides insights into sales, profit, customers, orders, regional performance, and monthly sales trends.

Features
KPI Cards
DAX Measures
Interactive Slicers
Sales by Category
Sales by Region
Monthly Sales Trend
Data Modeling
Power Query Data Cleaning
Tools Used
Power BI Desktop
Power Query
DAX
Excel
